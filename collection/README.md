# Collection

## Timetable

> [!NOTE]
> You must be within the base folder to be able to run the python scripts.

To rebuild the timetable database run `python -m collection.timetable.generate`.

## Stations

To generate the `stations` table in the database run the python script `python -m collection.stations.generate`

## Push Port

To replay the updates from the National Rail Darwin Push Port data run the python script `python -m collection.push-port.replay`
